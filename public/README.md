# public 资源目录说明

把需要上线的文件放在这里，路径与代码中的 `/xxx` 一一对应。

```
public/
├── models/          # 3D 化身 (.glb / .gltf)
│   └── avatar.glb
├── projects/        # 作品素材（按项目分子文件夹）
│   ├── nova-os/
│   │   └── cover.jpg
│   ├── flux-dashboard/
│   │   ├── demo.mp4
│   │   └── poster.jpg
│   └── arc-terminal/
│       ├── case-study.pdf
│       └── preview.jpg
├── gallery/         # 相册
│   ├── 01.jpg
│   └── reel.mp4
└── resume.pdf       # 简历 PDF
```

详细替换步骤见项目根目录 **CUSTOMIZATION.md**。
